Angular Assembly 6502 Practice

This work is based on the Making Games for the 2600 text by Steven Hugg. This repo mostly contains work starting with his samples and then adjusting to learn moving memory values between registers.

The contents of this repo are just for my projects and practice. Some of the late development of games for the Atari 2600 fascinate me. Developing cohesive gaming experiences with 128 bytes of ram, 0 video ram, and a limit of 4k (or so) makes for a impressive challenge to overcome. I maintain that the Double Dragon port for the Atari 2600 is the most technically impressive game I've seen.

Unity

In 2022 I started working with Unity. Initially I focused on learning the environment and the tools it provides. I'm building in a linux environment and some of the setup has proved challenging there. This initial project is to test a full project lifecycle: Starting from minor modifications and finishing with a published WebGl build.

There is nothing unique about this project. It is the demo 2d world sample provided by Unity. It is modified slightly following their tutorials, then built using the WebGl platform and published to unity's server. Unity provides an excellent suite of introductory videos to learn from and a set of projects to begin with.

I'm currently working on a 2d practice project to expand these skills a bit.
