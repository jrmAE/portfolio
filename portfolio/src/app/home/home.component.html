Joseph Meyer Sample Projects Assembly 6502 Practice

This work is based on the Making Games for the 2600 text by Steven Hugg. This repo mostly contains work starting with his samples and then adjusting to learn moving memory values between registers.

The contents of this repo are just for my projects and practice. Some of the late development of games for the Atari 2600 fascinate me. Developing cohesive gaming experiences with 128 bytes of ram, 0 video ram, and a limit of 4k (or so) makes for a impressive challenge to overcome. I maintain that the Double Dragon port for the Atari 2600 is the most techincally impressive game I've seen.

Assembly Basic Java-based Android Samples

This work contains samples that I use for an Android Development class that I teach. That class stresses the basics of the Android and assumes a entry-level familiarity with Java. We learn our way around Android Studio, learn the basics of version of control and some best practices for software development. We then cover topics that include the activity lifecycle, basic security principles and deployment. We use those skills to development an android application unique to each student then learn how to deploy them to our devices.

Permissions Location Animations Camera Basic Demo An Intro to Kotlin

This work contains samples that I use for an Android Development class that I teach. The class is 12 weeks long. For classes that move faster and can cover the critical development principles of Android in Java, we will begin an introduction-to and overview-of Kotlin.

By the time students are introduced to Kotlin, they would have completed a Java programming course in AndroidLocation to the bulk of their Android class being in Java. This provides an excellent base to compare and contrast the benefits of Kotlin.
